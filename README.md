# 🩺 AI Health Assistant

An AI-powered web application that helps users monitor their health through intelligent features such as symptom checking, BMI calculation, health articles, and an AI chatbot.

## ✨ Features

- 🤖 AI Health Chatbot
- 🩺 Symptom Checker
- 📊 BMI Calculator
- 📚 Health Articles
- 🔐 User Authentication
- 📱 Responsive Design
- 🌙 Dark & Light Mode

## 🛠️ Tech Stack

- React
- TypeScript
- Tailwind CSS
- Vite
- Bolt.new
- GitHub

## 🚀 Installation

```bash
git clone https://github.com/amirasamy19/Ai-health-assistant.git
cd Ai-health-assistant
npm install
npm run dev
```
